Hands-free voice workflow for Claude Code. Say "hey jarvis" with text highlighted anywhere and it's read aloud. Claude's replies are auto-spoken. Each terminal gets its own colour, shown as an emoji in its statusline and as a dot on the toolbar.

Global shortcuts

Global keyboard shortcuts
ShortcutAction
Ctrl+Shift+AShow / hide the toolbar
Ctrl+Shift+SRead the currently highlighted text
Ctrl+Shift+JToggle wake-word listening (chime confirms on/off)
Ctrl+Shift+PPause / resume playback
Ctrl+Shift+OPause only (without resuming on next clip)
Say "hey jarvis"Same as Ctrl+Shift+S on highlighted text

Tip: labels you set on a session show up next to its emoji in that terminal's statusline (e.g. 🟢 Frontend).

About the mascot: the walking character on the scrubber is a homage to Claude Code's thinking-spinner humour. The verbs that float above his head (Moonwalking, Finagling, Pontificating...) are the same list Claude Code itself uses while thinking. He only appears while a Claude Code response is playing; highlight-to-speak clips show a plain J badge instead. No affiliation with Anthropic ‚ÄĒ this is a fan project.

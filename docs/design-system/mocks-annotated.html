Playback section
Speed slider 0.5–2.5× · Auto-prune toggle + seconds (3–600). Flip auto-prune off to let clips stack up when you're stepping away.
Session row · 6 columns
Chevron · swatch · 8-char ID · editable label · colour dropdown (24 arrangements) · mute button.
Per-session mute
🔊 / 🔇 · muted rows fade to 55% · muted sessions skip synthesis entirely (no clips, no dots, no audio).
Expanded row
Chevron ▾ reveals the session's voice dropdown (45 Edge voices) and 6 tri-state speech-includes toggles: code blocks · inline code · URLs · headings · bullet markers · image alt-text.
Tri-state toggles
● on · ● default (follow global) · ○ off
Shortcuts
Three global hotkeys (A / S / J) plus local ones when the toolbar has focus.
